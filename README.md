# atom-echo-with-glassunit-sample

M5Stack Atom Echo with M5Stack GLASSUNIT sample.

`M5Unified.h` と `M5UnitGLASS.h` だけを使用した最低限のサンプル。

`M5UnitGLASS` の display関数を使用して、簡易な顔（スタックチャン）を描画する。

数秒ごとに、目と口が閉じたり空いたりする。
